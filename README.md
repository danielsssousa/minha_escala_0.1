#Ferramenta em Python para gestão automatizada de escalas de serviço, permitindo o registro de afastamentos (férias, dispensas e demais) e geração de relatórios.
 **Funcionalidades:**
- Gerenciamento de Efetivo: Cadastro de militares por graduação e antiguidade.

 - Controle de Afastamentos: Registro de férias, dispensas, LTS e outros motivos.

 - Lógica de Escala: Algoritmo que impede a escalação de militares afastados ou em folga.

 - Interface Intuitiva: Desenvolvida em Python para facilitar o uso por operadores da seção.
